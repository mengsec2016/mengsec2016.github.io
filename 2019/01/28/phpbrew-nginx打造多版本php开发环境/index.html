首先编译需要的PHP版本，在此我使用php-fpm来实现PHP和Nginx通信，需要在编译php时，添加+fpm参数。

3.1 编译php

在之前我已经配置好了php-5.6.40，在此我以php 7.2.14为例，再进行一次配置。
